Lab 10

Incorrect use of <label for=FORM_ELEMENT>

Solution (From Hilton Mai on the class discord): My labels used the "for" assignment to refer to the name attribute of the input field, which was throwing these issues. Once I changed the labels to refer to the id attribute of the form fields, the issues went away.
