Lab 10 - JavaScript Events and Forms

Challenge

The challenge was to use event listeners in Javascript to process and manipulate input and to manipulate the DOM to display these new values.

Problems

I had trouble figuring out how to add a name to the input. My solution was that I had to put a label tag to initialize the name variable then assign that to the input name next.

My first time testing tasks 2.1-2.3, index.html didn't display the results and thre the error: uName.split is not a function. My solution to this was adding type checking, but this wasn't the issue; I had accessed the entire element, and not its contents, so instead I accessed inputGet.value.

I also got the error: failed to execute write on Document (async error) on remote page. The solution was that I had to reload the page, as the JS code hadn't loaded into GitHub yet.
